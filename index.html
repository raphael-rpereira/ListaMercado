Compartilhar lista

Gere um link com a lista atual. Abra o arquivo HTML nos dois celulares e use o link para passar a lista de um para o outro.

A outra pessoa deve abrir o link no mesmo arquivo HTML salvo no celular dela

Como usar o compartilhamento
  1. Salve este arquivo HTML nos dois celulares
  2. Monte a lista em um celular
  3. Gere o link e envie pelo WhatsApp
  4. No outro celular, abra o link no navegador
  5. Copie o conteúdo após ?lista= e cole em:
    Menu → Importar lista
